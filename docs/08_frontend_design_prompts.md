# ITの右腕 — frontend-design スキル 共通プロンプトテンプレート

---

## 使い方

各セッションの**最初のプロンプトに貼り付けてから**指示を出す。
これをベースにすることでページをまたいでデザインが統一される。

---

## ✅ 共通制約（毎回先頭に貼る）

```
# ITの右腕 — デザイン共通制約

## サイト概要
中小企業・スタートアップ向けのDX・IT活用メディア。
記事で信頼を積み上げ、無料相談・問い合わせから案件獲得につなげる。

## ターゲット
中小企業の経営者・役員、現場の業務担当者、IT担当・情シス

## トーン・スタイル
editorial（メディアらしい洗練感）。信頼感・誠実さを優先しつつ、CTAへの誘導が強いデザイン。
派手すぎず、でも凡庸にならない。読み物として心地よい。

## カラー
- メイン: #1D9E75（グリーン）
- アクセント: #378ADD（ブルー）
- テキスト: #1a1a1a
- 背景: #ffffff / #f8f8f6（オフホワイト）

## タイポグラフィ
- 見出し: 個性のある日本語対応フォント（Noto Serif JP, Zen Old Mincho, Shippori Mincho 等）
- 本文: 読みやすさ優先（Noto Sans JP 等）
- 英字アクセント: 洗練系（Playfair Display, DM Serif Display 等）

## 技術スタック
- Next.js 14（App Router）
- Tailwind CSS
- Server Component ベース（"use client" は最小限）
- 画像は next/image を使用
- リンクは next/link を使用

## 共通コンポーネント
- Header: ロゴ / 記事一覧 / カテゴリ / 運営者紹介 / 無料相談ボタン / お問い合わせボタン
- Footer: 3カラム + 特商法・プライバシーポリシー
- CTAカラー: グリーン塗り（#1D9E75）がメインCTA、ブルーボーダーがサブCTA
```

```
## セクション構成（順番通りに）
1. ナビゲーション（Header コンポーネント）
2. ヒーロー: キャッチ「現場で使えるDX・IT活用のヒントが見つかる」/ サブ「難しい言葉なし。明日から使える情報だけ。」/ CTA2つ（無料相談30分 / 記事を読む）
3. 新着記事: 3カラムカード / サムネイル・カテゴリバッジ・タイトル・日付 /「一覧を見る →」リンク
4. カテゴリ一覧: 3カラム / アイコン・カテゴリ名・記事数
5. 中間CTA: グリーン背景 /「まず話してみませんか」/ 無料30分・オンライン・完全無料の3点
6. 運営者紹介: アイコン・名前・肩書き・スキルタグ（FastAPI / Next.js / SAP / DX支援）
7. Footer（Footer コンポーネント）
```

```
## フォームフィールド
- 名前（必須・最大50文字）
- 会社名（必須・最大100文字）
- メールアドレス（必須・メール形式）
- お問い合わせ内容（必須・最小10文字・最大1000文字）

## 要件
- React Hook Form + バリデーション
- 送信先: POST /api/contact
- 成功時: /contact/thanks へリダイレクト
- 失敗時: フォーム内にエラーメッセージ
- "use client" を ContactForm コンポーネントに付与
- ページ自体は Server Component
```

## Shift+Tab（プランモード）の活用

コードを書かせる前に `Shift+Tab` でプランモードに入り、
デザイン方向性を確認してから実装に進むと手戻りが減る。

```
# プランモードで確認すること
- フォントの選択が意図通りか
- カラーがブランドカラーと合っているか
- セクション構成の順序が正しいか
- モバイルレイアウトの方針
```
